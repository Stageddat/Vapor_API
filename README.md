# Vapor API - Laravel CRUD
CRUD d'una botiga de videojocs inspirada en Steam de Valve.<br>
DAM 1M Prova competencial AEA5: Bases de dades documentals.

Fet amb el framework de PHP Laravel 13 juntament amb TailwindCSS, Artisan i SQLite.

Les dades, informació i imatges són extractes de [IGDB](https://www.igdb.com/). <br>
Steam i el logotip de Steam són marques comercials de Valve Corporation. Totes les altres marques comercials són propietat dels seus respectius propietaris.
